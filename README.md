# How similar are MLP hidden states to each other?

A systematic study of cross-layer representation similarity in
feed-forward ReLU multi-layer perceptrons, trained on CIFAR-10 and
MNIST with varying (depth, width).

Generated by NeuriCo on 2026-04-24.

## TL;DR

Across (depth, width) configurations, MLP hidden states at different
layers are **neither identical nor totally different** — mean
off-diagonal linear-CKA sits consistently in (0.5, 0.95). Three
findings stand out:

1. **Depth produces block structure in pure MLPs.** At depth 16 the
   middle layers form a near-uniform block with CKA ≳ 0.9 — the first
   clean demonstration of the block-structure phenomenon (previously
   reported for CNNs, ResNets, transformers) in a pure MLP.
2. **Width *decreases* cross-layer CKA.** Narrow MLPs have more
   similar consecutive layers (less per-layer transformation
   capacity); wider MLPs allow each layer to transform the
   representation more aggressively (Spearman ρ = −0.95 at depth 4 or
   8 on CIFAR-10).
3. **Cross-metric agreement.** Linear CKA, orthogonal Procrustes, and
   sample-wise cosine similarity all rank layer pairs in the same
   order (ρ_CKA-Proc = 0.98; ρ_CKA-cos = 0.72), so the findings are
   robust to the Davari-et-al.-2022 critique of CKA.

See `REPORT.md` for the full write-up and `figures/` for all plots.

## Reproduce

```bash
uv venv && uv sync
# ensure a CUDA 12.x PyTorch wheel (we used torch 2.5.1+cu124)

# main (depth × width × seed) sweep:
python -m src.run_sweep --datasets cifar10 mnist \
    --depths 4 8 16 --widths 128 512 2048 --seeds 0 1 2 \
    --cifar-epochs 15 --mnist-epochs 8 --batch-size 512

# rich/lazy, shuffled-label, and dominant-datapoint ablations:
python -m src.ablations --dataset cifar10 --depth 8 --width 512

# figures + hypothesis tests:
python -m src.analysis
```

## Key hypotheses tested

| # | Hypothesis | Result |
|---|---|---|
| H1 | Adjacent-layer CKA ≫ far-layer CKA | **Supported** (paired t, p ≪ 10⁻³ on both datasets) |
| H2 | Wider MLPs → higher cross-layer CKA | **Refuted** — wider MLPs have *lower* cross-layer CKA (opposite sign to Kornblith 2019's cross-model finding) |
| H3 | Deeper MLPs develop block structure | **Supported** (qualitative heatmaps at d=16) |
| H4 | CKA / cosine / Procrustes agree on ranking | **Supported** (ρ = 0.98 with Procrustes, 0.72 with cosine) |
| H5 | Lazy regime → higher CKA | **Supported** (Spearman ρ = −0.40 on CIFAR-10, p = 0.048) |
| H6 | Dominant datapoints explain block CKA | **Weakly supported** — dropping top-decile high-norm samples reduces CKA by only 0.011 |
